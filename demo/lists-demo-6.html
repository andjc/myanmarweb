Sample code:


    #main-content h1:lang(my):before {
            content: counter(chapt, myanmar) "။" ;
            padding-right: 0.5em;
            color: #000000;
    }
    
    #main-content h1:lang(my) {
            counter-increment: chapt;
            counter-reset: section;
    }
   
    #main-content h2:lang(my):before {
            content: counter(chapt, myanmar) "." counter(section, myanmar) "။" ;
            padding-right: 0.5em;
            color: #000000;
    }
    
    #main-content h2:lang(my) {
            counter-increment: section;
            margin: 1.0em;
    }

    #main-content h3:lang(my):before {
            content: counter(chapt, myanmar) "." counter(section, myanmar) "." counter(subsection, myanmar) "။" ;
            padding-right: 0.5em;
            color: #000000;
    }
    
    #main-content h3:lang(my) {
            counter-increment: subsection;
            margin: 1.0em;
    }

    :lang(my) {font-family Padauk, Tharlon, sans-serif;}

Expected behaviour: Header(h1...6) are prefixed with "Burmese Digits.
